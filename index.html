Começando

Criar um arquivo.html com a seguinte estrutura

<!-- Pode-se utilizar o comando para realizar comentários dentro do arquivo -->

para que o comando acima não seja interpretado como parte do HTML, é necessário que ele esteja assim:
&lt; <!-- comentário --> &gt; Aliás, reparou na quebra de linha?
Pois é, não é algo entendido automaticamente, é um comando representado por <br>
